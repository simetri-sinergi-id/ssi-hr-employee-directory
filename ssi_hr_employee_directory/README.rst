==================
Employee Directory
==================

This module adds module HR Employee Directory.

Installation
============

To install this module, you need to:

1.  Clone the branch 14.0 of the repository https://github.com/simetri-sinergi-id/ssi-hr-employee-directory
2.  Add the path to this repository in your configuration (addons-path)
3.  Update the module list
4.  Go to menu *Setting -> Modules -> Local Modules*
5.  Search For *Employee Directory*
6.  Install the module
